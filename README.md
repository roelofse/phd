## Governance (T1–T4 in one paragraph)
T1–T4 is a **feedback spiral**, not a pipeline:
- **T1 Truthing** surfaces localized friction/anomalies (+R, +D).
- **T2 Trusting** validates constraints and permissioning (+R, +C).
- **T3 Transforming** engineers auditable interventions (+S, +C).
- **T4 Transmitting** delivers outcomes + recursion logs (+S, +D), feeding back to T1.

Momentum (+L) regulates pace to prevent drift into **wilderness** (endless sensing) or **factorization** (premature optimization).
